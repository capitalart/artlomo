================================================================================

   - LISTING_BOILERPLATE
   - TECHNICAL_EXCELLENCE_BLOCK
   - build_seed_context() if metadata exists

2. Load prompts.py system instructions
   - HERITAGE_FIRST_SYSTEM_PROMPT_TEMPLATE
   - Output format requirements
   - Cultural heritage protocol

3. Load instructions/*.md examples
   - MASTER_ETSY_DESCRIPTION_ENGINE.md
   - MASTER-ARTWORK-ANALYSIS-LISTING-DESCRIPTION-EXAMPLE.md

4. Send to AI Provider (OpenAI GPT-4o or Gemini Pro Vision)

5. Parse structured JSON response

6. Post-Processing Pipeline:
   Step 1: _prepend_functional_hook() (160-250 chars)
   Step 2: _inject_gold_standard_blocks()
   Step 3: Normalize materials (13 buyer-focused items)
   Step 4: Sanitize tags (max 20 chars, 13 items)
   Step 5: Generate SEO filename (66-char limit)

7. Save listing.json

8. User review at /artwork/{slug}/review/{provider}
