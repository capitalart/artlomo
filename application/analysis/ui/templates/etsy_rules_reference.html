Etsy Rules Reference

Field-by-field listing rules for Analysis Review pages. Use anchor links from each field’s “Etsy Rules” link.

Title Description Tags Materials Section Quantity Price

Title

  • Maximum 140 characters.
  • Front-load with at least 3 high-intent keywords.
  • Avoid all-caps titles.
  • Keep language clear, descriptive, and human-readable.

Description

  • The first 160 characters should carry the strongest SEO intent.
  • Include a clear Materials section.
  • Include a clear Shipping section.
  • Keep structure scannable with short paragraphs or bullets.

Tags

  • Use exactly 13 tags.
  • Each tag must be 20 characters max.
  • No duplicate tags (case-insensitive).
  • Use buyer-language, not internal studio shorthand.

Materials

  • Maximum 13 material entries.
  • Each material entry must be 45 characters max.
  • Use concise terms buyers recognize.

Section

  • Assign listings to a store section matching the artwork intent.
  • Use one consistent taxonomy to keep catalog filtering clean.
  • Avoid misc/catch-all sections for publish-ready listings.

Quantity

  • Required for Etsy publish compliance.
  • Must be a positive whole number.

Price

  • Required for Etsy publish compliance.
  • Must be a valid positive amount in store currency.
  • Prefer clear, consistent pricing with your collection strategy.
