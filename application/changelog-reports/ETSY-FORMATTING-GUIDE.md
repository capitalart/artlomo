# ✅ ETSY DESCRIPTION FORMATTING GUIDE (Plain Text Only)

**Date:** March 3, 2026
**Status:** UPDATED - Etsy-Safe Plain Text Formatting

---

## 🎯 **CRITICAL RULE: No HTML, No Markdown**

Etsy **strips all HTML and markdown** from descriptions. Every description must be **plain text**.

### What THIS Means

- ❌ Don't use `<h1>`, `<br>`, `<div>` (HTML tags)

- ❌ Don't use `###`, `**bold**`, `~~strikethrough~~` (Markdown syntax)

- ❌ Don't use backticks or code blocks

- ✅ Use emojis, line breaks, and Unicode characters only

---

## ✅ **ALLOWED FORMATTING ELEMENTS (Etsy-Safe)**

### 1. **Emojis** ✨

Used to visually separate sections and guide the reader's eye.

```text
🌅 [Section Header]
🌿 [Section Header]
💎 [Section Header]
📐 [Section Header]
❤️ [Section Header]
```

**Why:** Emojis are readable on ALL devices (mobile, desktop, email). They provide visual hierarchy without HTML.

---

### 2. **Line Breaks (Blank Lines)**

Press Enter twice to create a blank line between sections.

## CORRECT

```text
First section text.

New section starts here.
```

## INCORRECT

```text
First section text.
New section starts here.
```

**Why:** Blank lines are crucial for mobile readability (60%+ of Etsy traffic).

---

### 3. **Unicode Characters for Visual Breaks**

**Em-Dash (—)** for sophisticated separation:

```text
Boandik (Bunganditj)—the People of the Reeds
```

**Horizontal Lines** using dashes or asterisks:

```text
---
```

OR

```text
***
```

**Bullet Points** using Unicode:

```text
• Item one
• Item two
• Item three
```

OR

```text
◆ Item one
◆ Item two
```

OR

```text
★ Item one
★ Item two
```

---

### 4. **Numbered Lists**

For step-by-step instructions:

```text
1. Add to Cart
2. Download file
3. Print at home
4. Display proudly
```

---

### 5. **ALL CAPS for Emphasis**

Instead of bold or italics, use all caps for important terms (since Etsy may strip formatting):

```text
You'll receive a museum-quality, ultra-high definition JPG at 14,400px and 300 DPI.
```

OR emphasize with context:

```text
This file is engineered for MASSIVE SCALE PRINTING up to 48 inches (121.9 cm).
```

## 🚫 **WHAT NOT TO DO**

### ❌ DON'T DO THIS

```html
<h3>The Soul of the Piece</h3>
<p>Immerse your space in the sun-drenched glow of the limestone coast.</p>

<ul>
<li>Oversized Clarity</li>
<li>Infinite Detail</li>
</ul>
```

**Why:** Etsy strips ALL HTML tags. Description will look like raw code.

---

❌ DON'T DO THIS

```markdown

### The Soul of the Piece

Immerse your space in the sun-drenched glow of the limestone coast.

- Oversized Clarity
- Infinite Detail
```

**Why:** Etsy doesn't render markdown. Will appear as raw `###` and `-` characters.

---

❌ DON'T DO THIS

```text
Boandik (Bunganditj)-the People of the Reeds
```

**Why:** Single dash looks unprofessional. Use em-dash (—) for gallery aesthetic.

## ✅ **DESCRIPTION QUALITY CHECKLIST**

Before submitting each Etsy listing:

- [ ] **Plain text ONLY** (no HTML, no markdown)

- [ ] **Emojis used** to separate major sections

- [ ] **Blank lines between paragraphs** for readability

- [ ] **Em-dashes used** (—) not hyphens

- [ ] **Heritage statement includes:** `Boandik (Bunganditj)—the People of the Reeds`

- [ ] **14,400px and 48-inch emphasized** as key value propositions

- [ ] **Minimum 850 characters** (Google SEO requirement)

- [ ] **Exactly 13 tags** (max 20 chars each)

- [ ] **Ends with:** "Warm regards, Robin Custance"

- [ ] **No quotation marks** (use single quotes ' if needed)
